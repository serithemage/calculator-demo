# Product Requirements Document (PRD)
# Scientific Calculator Web Application

## 1. 제품 개요

### 1.1 제품명
**Scientific Calculator** (공학용 전자계산기)

### 1.2 제품 비전
모바일 친화적이고 현대적인 디자인의 웹 기반 공학용 계산기를 제공하여, 학생, 엔지니어, 연구원 등이 언제 어디서나 복잡한 수학 계산을 쉽게 수행할 수 있도록 지원합니다.

### 1.3 목표 사용자
- 고등학생 및 대학생 (수학, 과학, 공학 전공)
- 엔지니어 및 연구원
- 일상적으로 복잡한 계산이 필요한 전문가
- 빠른 계산 도구가 필요한 모든 사용자

### 1.4 핵심 가치 제안
- **접근성**: 웹 기반으로 별도 설치 없이 브라우저에서 즉시 사용 가능
- **현대적 UI/UX**: 다크모드 지원 및 직관적인 인터페이스
- **모바일 최적화**: 반응형 디자인으로 모든 디바이스에서 원활한 사용
- **공학용 기능**: 삼각함수, 로그, 지수 등 고급 수학 함수 지원

## 2. 기능 요구사항

### 2.1 핵심 기능 (MVP)

#### 2.1.1 기본 산술 연산
- **사칙연산**: 덧셈(+), 뺄셈(−), 곱셈(×), 나눗셈(÷)
- **소수점 입력**: 소수점(.) 지원
- **괄호 연산**: 괄호를 이용한 연산 우선순위 제어 `( )`
- **퍼센트 계산**: 백분율(%) 연산

#### 2.1.2 공학용 함수
**삼각함수**
- `sin`: 사인 함수
- `cos`: 코사인 함수
- `tan`: 탄젠트 함수

**로그 함수**
- `ln`: 자연로그 (밑이 e인 로그)
- `log`: 상용로그 (밑이 10인 로그)

**기타 수학 함수**
- `1/x`: 역수 계산
- `√`: 제곱근
- `^`: 거듭제곱

**상수**
- `π` (파이): 원주율 (3.14159...)
- `e`: 자연상수 (2.71828...)

#### 2.1.3 각도 단위 설정
- **DEG (Degree)**: 도 단위 (기본값)
- **RAD (Radian)**: 라디안 단위
- 토글 스위치로 간편하게 전환 가능

#### 2.1.4 입력 및 편집
- **숫자 입력**: 0-9 숫자 버튼
- **백스페이스**: 마지막 입력 문자 삭제 (아이콘 버튼)
- **전체 삭제 (AC)**: All Clear - 모든 입력 및 결과 초기화
- **실시간 수식 표시**: 입력 중인 수식을 상단에 표시

#### 2.1.5 결과 표시
- **현재 결과**: 큰 폰트로 계산 결과 표시 (5xl-6xl 크기)
- **이전 수식**: 작은 폰트로 입력된 수식 표시 (투명도 60%)
- **천 단위 구분**: 숫자에 콤마(,) 표시로 가독성 향상

#### 2.1.6 히스토리 기능
- **계산 기록**: 이전 계산 내역 확인 (히스토리 아이콘 버튼)
- 히스토리 패널에서 이전 계산 재사용 가능

### 2.2 UI/UX 요구사항

#### 2.2.1 디자인 시스템
**컬러 팔레트**
- Primary Color: `#136dec` (파란색)
- Background Light: `#f6f7f8` (밝은 회색)
- Background Dark: `#101822` (진한 남색)
- Surface Dark: `#1b2531` (버튼 배경)
- Surface Dark Hover: `#25303e` (버튼 호버)

**타이포그래피**
- Display Font: `Space Grotesk` (헤더, 버튼)
- Body Font: `Noto Sans` (본문, 숫자)

**아이콘**
- Material Symbols Outlined 사용

#### 2.2.2 레이아웃
- **최대 너비**: 448px (max-w-md) - 모바일 중심 디자인
- **중앙 정렬**: 화면 중앙에 계산기 배치
- **그림자 효과**: shadow-2xl로 입체감 부여

**구성 요소**
1. **헤더**: 
   - 중앙: 모드 표시 ("Standard")
   - 우측: 히스토리 버튼
   
2. **디스플레이 영역**:
   - 이전 수식 (작은 텍스트, 투명도 60%)
   - 현재 결과 (큰 텍스트, 볼드)
   
3. **컨트롤 영역**:
   - DEG/RAD 토글 스위치
   - 백스페이스 버튼
   - 공학용 함수 그리드 (5열 × 2행)
   - 구분선
   - 메인 키패드 (4열 × 5행)

#### 2.2.3 버튼 스타일
**공학용 함수 버튼**
- 크기: h-10 (40px)
- 배경: 밝은 회색 / 다크 서페이스
- 간격: gap-2.5

**숫자 버튼 (0-9, .)**
- 크기: h-16 (64px)
- 배경: 흰색 / 다크 서페이스 (#25303e)
- 둥근 모서리: rounded-2xl
- 그림자: shadow-sm

**연산자 버튼 (+, −, ×, ÷, %, ( ))**
- 크기: h-16 (64px)
- 배경: 밝은 회색 / 다크 서페이스
- 텍스트 색상: Primary Blue

**특수 버튼**
- **AC**: 빨간색 텍스트 (dark:text-red-400)
- **=**: Primary 배경, 흰색 텍스트, 2열 너비, 그림자 효과

#### 2.2.4 인터랙션
- **호버 효과**: brightness-110 또는 배경색 변경
- **클릭 효과**: scale(0.96) - 버튼 눌림 애니메이션
- **전환 효과**: transition-all로 부드러운 상태 변화
- **반응형**: 터치 및 마우스 입력 모두 지원

#### 2.2.5 다크모드
- **기본값**: 다크모드 활성화 (`class="dark"`)
- **자동 전환**: 시스템 설정 또는 수동 토글 지원 (향후 기능)
- **색상 대비**: WCAG 접근성 기준 준수

### 2.3 기술 요구사항

#### 2.3.1 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS Framework**: Tailwind CSS

## 3. 사용자 시나리오

### 3.1 기본 계산
```
사용자: 학생
목표: 1,234.56 + sin(30°) 계산

1. 계산기 웹앱 접속
2. "1234.56" 입력
3. "+" 버튼 클릭
4. "sin" 버튼 클릭
5. "30" 입력
6. "=" 버튼 클릭
7. 결과 확인: 1,235.06
```

### 3.2 각도 단위 변경
```
사용자: 엔지니어
목표: 라디안 단위로 삼각함수 계산

1. DEG/RAD 토글에서 "RAD" 선택
2. "sin" 버튼 클릭
3. "π" 버튼 클릭
4. "÷" 버튼 클릭
5. "2" 입력
6. "=" 버튼 클릭
7. 결과 확인: 1.0
```

### 3.3 복잡한 수식
```
사용자: 연구원
목표: (5 + 3) × 2^3 ÷ √16 계산

1. "(" 버튼 클릭
2. "5 + 3" 입력
3. ")" 버튼 클릭
4. "×" 버튼 클릭
5. "2" 입력
6. "^" 버튼 클릭
7. "3" 입력
8. "÷" 버튼 클릭
9. "√" 버튼 클릭
10. "16" 입력
11. "=" 버튼 클릭
12. 결과 확인: 16
```

## 4. 비기능 요구사항

### 4.1 신뢰성
- **정확도**: IEEE 754 부동소수점 표준 준수
- **오류 처리**: 0으로 나누기, 정의되지 않은 연산 등 예외 처리
- **입력 검증**: 유효하지 않은 수식 입력 방지

### 4.2 사용성
- **직관성**: 첫 사용자도 설명 없이 사용 가능
- **피드백**: 모든 버튼 클릭에 시각적 피드백 제공
- **오류 메시지**: 명확하고 이해하기 쉬운 오류 안내

### 4.3 유지보수성
- **코드 품질**: ESLint, Prettier 적용
- **문서화**: 주요 함수 및 컴포넌트 주석 작성
- **테스트**: **Core Logic** 단위 테스트 커버리지 100% (UI 자동화 테스트 제외)

### 4.4 확장성
- **모듈화**: 기능별 컴포넌트 분리
- **플러그인**: 향후 추가 함수 쉽게 통합 가능
- **테마**: 커스텀 테마 지원 가능한 구조
